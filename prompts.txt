sur une nouvelle branche je veux réorganiser l'UX :

A/ 1/le user doit pouvoir noter ses activités tous les jours grâce à un agenda: tu dois rajouter un volet calendrier à l'UI, le user peut rajouter une activité cliquant sur un des jours du calendrier
2/ Une activité est effectuée sur une ou plusieurs zones par un ou plusieurs ouvriers, 
3/ Les activités sont enregistré dans l'historique des zones dans la partie engrais ou arrosage (si le user précise que c'est une activité de type engrais ou arrosage) avec un nombre d'ouvriers
4/ Le user peut aussi enregistrer une activité "Autres" et la renseigner grâce à un text input, cette activité est aussi renseignée dans l'historique de/des zones sélectionnées
5/ Le user pourra modifier cette activité une fois enregistré, soit depuis le calendrier, soit depuis le détail de l'historique de la zone une modification de date modifiera la place de cette activité dans le calendrier



B/ Lors de l'ajout d'un point avec le bouton en Screenshot ci-joint, deux input apparaissent pour les coordonnées GPS, le user a alors le choix entre cliquer sur la map ou rentrer les coordonnées GPS à la main : si le point est hors du champs alors un message d'avertissement s'affiche et le point n'est pas enregistré, le user peut ajouter des notes à ce point 



C/ 1/ Une zone doit pouvoir être archivée: Si le user décide d'archiver une zone alors toutes informations de suivis de cette zone sont grisées dans le calendrier et reste en base mais sont marquées comme archivées. 
2/ Les coordonnées de détour de la zone archivée sont conservées dans une liste en bas de la sidebar de droite ("Archives") mais cette zone n'est plus modifiable, il faut la désarchiver pour pouvoir la modifier cependant le user doit pouvoir la visualiser sur la map avec un bouton afficher/masquer sans pour autant la désarchiver
3/ Lors de l'archivage, le user doit pouvoir réattribuer les différents items de l'historique de cette zone archivée à une ou plusieurs zone mais ça ne les efface pas de l'historique de cette zone archivée,
